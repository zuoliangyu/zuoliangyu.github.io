事件的运算及概率

事件及运算

知识点

  • 包含事件
    • $A$事件是$B$事件中的一部分
    • $$A \subset B$$
  • 并（和）事件
    • $A$事件与$B$事件的集合
    • $$A \cup B=A+B$$
  • 差事件
    • $A$事件中会发生，而$AB$的交集事件不会发生
    • $A-B=A \bar B=A-AB$
  • 交（积）事件
    • $A$事件与$B$事件的交集部分
    • $A \cap B=AB$
  • 互斥事件
    • $A$事件与$B$事件没有交集$\to$不会同时发生
    • $$AB = \emptyset $$
  • 对立（逆）事件
    • 在总事件的集合$S$中不属于$A$事件的部分
    • $A \cup \bar A=S,A\bar A=\emptyset$
  • 独立事件
    • $AB$在不同点事件框架中
    • $P(AB)=P(A)P(B)$
    • $AB$独立$\to A \bar B,\bar A B,\bar A \bar B$独立
    • 若$ABC$相互独立
      • 两两独立
      • $P(ABC)=P(A)P(B)P©$
    • 两两独立$\nrightarrow ABC$相互独立
  • 公式
    • 德摩根律
      • 长杠变短杠，开口换方向
      • $$\overline {A \cup B} = \overline A \cap \overline B=\overline A \cdot \overline B $$
      • $\overline {A \cap B}=\overline A \cup \overline B=\overline A+\overline B$
    • 加法公式
      • $P(A\cup B)=P(A)+P(B)-P(AB)$
      • $P(A\cup B \cup C)=P(A)+P(B)+P©-P(AB)\-P(AC)-P(BC)+P(ABC)$
    • 减法公式
      • $P(A -B)=P(A\overline B)=P(A)-P(AB)$
    • 对立事件
      • $P(\overline A)=1-P(A)$
    • 独立事件
      • $P(AB)=P(A)P(B)$

    • 在$A$发生的情况下发生事件$B$

  • 乘法公式
    • $P(AB)=P(A)P(B|A)=P(B)P(A|B)$

全概率、贝叶斯公式

知识点

  • 设$A$为发生的事件
  • 找出完备事件组$B_i$
  • 写出$P(B_i),P(A|B_i)$
  • 带入全概率公式
    • $$P\left( A \right) = \sum\limits_{i = 1}^n {P\left( { {B_i} } \right)P\left( {A|{B_i} } \right)} $$
  • 贝叶斯公式
    • $P(B_i|A)=\frac{P(B_i)P(A|B_i)}{P(A)}$

概率密度，分布函数

知识点

  • 概率密度$f(x)$
    • $$\int_{ - \infty }^{ + \infty } {f\left( x \right)dx} = 1$$
    • $$F\left( x \right) = P\left{ {X \leqslant x} \right} = \int_{ - \infty }^x {f\left( x \right)dx} $$
    • $$P\left{ {a \leqslant X \leqslant b} \right} = F\left( b \right) - F\left( a \right) = \int_a^b {f\left( x \right)dx} $$
  • 分布函数$F(x)$
    • $0 \leqslant F(x) \leqslant 1$
    • $F’(x)=f(x)$
    • $F(- \infty)=0,F(\infty)=1$
    • $F(X)$单调不减函数
    • 右连续$$\mathop {\lim }\limits_{x \to x_0^ + } F\left( x \right) = F\left( { {x_0} } \right)$$

泊松分布

知识点

  • 符号表达
    • $$X \sim \pi \left( \lambda \right)$$或$X\sim P(\lambda )$
  • 分布律
    • $$P\left{ {X = k} \right} = \frac{ { {\lambda ^k} } }{ {k!} }{e^{ - \lambda } }$$
  • 可加性推导
    • $$\eqalign{
      & X \sim \pi \left( \lambda \right) \to P\left{ {X = k} \right} = \frac{ { {\lambda ^k} } }{ {k!} }{e^{ - \lambda } } \cr
      & Y \sim \pi \left( \mu \right) \to P\left{ {Y = k} \right} = \frac{ { {\mu ^k} } }{ {k!} }{e^{ - \mu } } \cr
      & Z = X + Y \cr
      & P\left{ {Z = k} \right} = \sum\limits_{i = 0}^k {P\left( {X = i} \right)P\left( {Y = k - i} \right)} = \sum\limits_{i = 0}^k {\frac{ { {\lambda ^k} } }{ {k!} }{e^{ - \lambda } }\frac{ { {\mu ^{\left( {k - i} \right)} } } }{ {\left( {k - i} \right)!} }{e^{ - \mu } } } \cr
      & = \frac{ { {e^{ - \left( {\lambda + \mu } \right)} } } }{ {k!} }\sum\limits_{i = 0}^k {\frac{ {k!} }{ {i!\left( {k - i} \right)!} }{\lambda ^k}{\mu ^{k - i} } } = \frac{ { {e^{ - \left( {\lambda + \mu } \right)} } } }{ {k!} }C_k^i\left( { {\lambda ^k}{\mu ^{k - i}} } \right) = \frac{ { {e^{ - \left( {\lambda + \mu } \right)} } } }{ {k!} }{\left( {\lambda + \mu } \right)^k} \cr
      & \Rightarrow Z \sim \pi \left( {\lambda + \mu } \right) \cr} $$

正态分布

知识点

  • 符号表达
    • $$X \sim N\left( {\mu ,{\sigma ^2} } \right)$$
  • 概率密度
    • $$f\left( x \right) = \frac{1}{ {\sqrt {2\pi } \sigma } }{e^{ - \frac{ { { {\left( {x - \mu } \right)}^2} } }{ {2{\sigma ^2} } } } }$$
  • 图像
    • 关于$\mu$对称
    • $\sigma $越小，图像越陡（集中）
  • 标准正态分布
    • $X\sim N(0,1)$
    • $$\Phi \left( x \right) = P\left{ {X \leqslant x} \right} = \int_{ - \infty }^x {f\left( x \right)dx} $$
    • 概率密度
      • $f(x)=\frac {1}{\sqrt{2 \pi}}e^{-\frac{x^2}{2}}$
    • 性质
      • $$\mu = 0,\Phi \left( 0 \right) = \frac{1}{2}$$
      • $$\Phi \left( { - x} \right) = 1 - \Phi \left( x \right)$$
      • 若$$X \sim N\left( {\mu ,{\sigma ^2} } \right)$$
        • $$\frac{ {X - \mu }}{\sigma } \sim N\left( {0,1} \right)$$
        • $$P\left{ {X < a} \right} = \Phi \left( {\frac{ {a - \mu } }{\sigma } } \right)$$
        • $$P\left{ {b < X < a} \right} = \Phi \left( {\frac{ {a - \mu } }{\sigma } } \right) - \Phi \left( {\frac{ {b - \mu } }{\sigma } } \right)$$
  • 可加性推导
    • $$\eqalign{
      & X \sim N\left( { {\mu _1},\sigma _1^2} \right) \cr
      & Y \sim N\left( { {\mu 2},\sigma 2^2} \right) \cr
      & Z = X + Y \cr
      & {f_Z}\left( z \right) = \int      { - \infty }^{ + \infty } { {f_X}\left( x \right){f_Y}\left( {z - x} \right)dx} \cr
      & = \int      { - \infty }^{ + \infty } {\frac{1}{ {\sqrt {2\pi } {\sigma _1} } }{e^{ - \frac{ { { {\left( {x - {\mu _1} } \right)}^2} } }{ {2\sigma _1^2} } } }\frac{1}{ {\sqrt {2\pi } {\sigma _2} } }{e^{ - \frac{ { { {\left( {z - x - {\mu _2} } \right)}^2} } }{ {2\sigma _2^2} } } }dx} \cr
      & = \frac{1}{ {2\pi {\sigma _1}{\sigma 2} } }\int{ - \infty }^{ + \infty } { {e^{ - \left[ {\frac{ { { {\left( {x - {\mu _1} } \right)}^2} } }{ {2\sigma _1^2} } + \frac{ { { {\left( {z - x - {\mu _2} } \right)}^2} } }{ {2\sigma _2^2} } } \right] } }dx} \cr
      & = \frac{1}{ {2\pi {\sigma _1}{\sigma 2} } }\int{ - \infty }^{ + \infty } { {e^{ - \left( {a{x^2} - 2bx + c} \right)} }dx} \cr
      & a = \frac{1}{2}\left( {\frac{1}{ {\sigma _1^2} } + \frac{1}{ {\sigma _2^2} } } \right),b = \frac{1}{2}\left( {\frac{ { {\mu _1} } }{ {\sigma _1^2} } + \frac{ {z - {\mu _2} } }{ {\sigma _2^2} } } \right),c = \frac{1}{2}\left[ {\frac{ {\mu _1^2} }{ {\sigma _1^2} } + \frac{ { { {\left( {z - {\mu _2} } \right)}^2} } }{ {\sigma 2^2} } } \right] \cr
      & \int      { - \infty }^{ + \infty } { {e^{ - \left( {a{x^2} - 2bx + c} \right)} }dx} = \sqrt {\frac{\pi }{a} } {e^{\frac{ {ac - {b^2} } }{a} } } \cr
      & \Rightarrow {f_Z}\left( z \right) = \frac{1}{ {2\pi {\sigma _1}{\sigma 2} } }\int{ - \infty }^{ + \infty } { {e^{ - \left( {a{x^2} - 2bx + c} \right)} }dx} = \frac{1}{ {\sqrt {2\pi } {\sigma _1}{\sigma _2} } }{e^{\frac{ { { {\left[ {z - \left( { {\mu _1} + {\mu _2} } \right)} \right]}^2} } }{ {2\left( {\sigma _1^2 + \sigma _2^2} \right)} } } } \cr
      & \Rightarrow Z \sim N\left( { {\mu _1} + {\mu _2},\sigma _1^2 + \sigma _2^2} \right) \cr} $$

知识点

  • 概率密度
    • $$\int_{ - \infty }^{ + \infty } {\int_{ - \infty }^{ + \infty } {f\left( {x,y} \right)dx} dy} = 1$$
    • $$P\left{ {\left( {X,Y} \right) \in D} \right} = \iint\limits_D {f\left( {x,y} \right)dy}$$
  • 边缘概率密度
    • $${f_X}\left( x \right) = \int_{ - \infty }^{ + \infty } {f\left( {x,y} \right)dy} $$
    • $${f_Y}\left( y \right) = \int_{ - \infty }^{ + \infty } {f\left( {x,y} \right)dx} $$
  • 条件概率密度
    • $${f_{X|Y} }\left( {x|y} \right) = \frac{ {f\left( {x,y} \right)} }{ { {f_Y}\left( y \right)} }$$
    • $${f_{Y|X} }\left( {y|x} \right) = \frac{ {f\left( {x,y} \right)} }{ { {f_X}\left( x \right)} }$$
  • 独立性
    • $$f\left( {x,y} \right) = {f_X}\left( x \right){f_Y}\left( y \right)$$

知识点

  • 常用统计量

三种常见分布

知识点

  • 卡方分布（$${\chi ^2}$$分布）
    • 若$${X_1},{X_2}, \cdots ,{X_n}$$独立且服从$N(0,1)$,则$$X_1^2 + X_2^2 + \cdots + X_n^2 \sim {\chi ^2}\left( n \right)$$
    • 性质
      • $X\sim \chi^2(n_2),Y\sim \chi^2(n_2)\to X+Y\sim \chi^2(n_1+n_2)$
  • $t$分布
    • 若$X\sim N(0,1),Y\sim \chi^2(n)$且$XY$相互独立，则$\frac{X}{\sqrt{Y/n } }\sim t(n)$
  • F分布
    • $X\sim \chi^2(n_2),Y\sim \chi^2(n_2)$且$XY$相互独立，则$\frac{X/n_1}{Y/n_2}\sim F(n_1,n_2)$
    • 性质
      • $F(n_1,n_2)=\frac{1}{F(n_2,n_1)}$
      • $F_{1-a}(n_1,n_2)=\frac{1}{F_a(n_2,n_1)}$
