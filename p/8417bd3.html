逻辑代数

逻辑运算及逻辑门

知识点

  • 偶数个1异或结果为0，奇数个1异或结果为1，无论多少个0异或结果都为0。
  • 偶数个0同或结果为1，奇数个0同或结果为0，无论多少个1同或结果都为1。
  • 普通门电路只能输出高电平和低电平，而三态门可多输出一种状态：高阻态。

  • 集电极开路与非门（OC门），输出端看直接相连完成线与逻辑功能

解题（逻辑运算及逻辑门）

规则及常用形式

  • 反演规则
    • 与项加括号
    • 与或项交换
    • 短非号交换
    • 长非号不动
    • $$F = \overline {A\overline C + \overline {BC} } \Rightarrow \overline F = \overline {\left( {\overline A + C} \right) \cdot \overline {\overline B + \overline C } } $$
  • 对偶规则
    • 与项加括号
    • 与或项交换
    • 长非号不动
    • $$F = \overline {A\overline C + \overline {BC} } \Rightarrow F’ = \overline {\left( {A + \overline C } \right) \cdot \overline {B + C} } $$
  • 与非——与非表达式
    • 化为与或
    • 加入2个长非
    • 利用反演规则
    • $$F = A\left( {B + C} \right) + BC = AB + AC + BC = \overline{\overline {AB + AC + BC} } = \overline {\overline {AB} \cdot \overline {BC} \cdot \overline {AC} } $$
  • 最小项$m$
    • 包含所有输入的与项，$n$个变量的最小项有$2^n$个
    • 两个变量可以构成四个最小项
      • $$AB,A\bar B,\bar AB,\bar A\bar B$$

触发器

触发方式

知识点

  • 当它被特定信号触发时，输出发生变化。触发器的输出是由当前状态和当前输入决定的。
  • 触发方式
    • 电平触发
    • 主从触发
    • 边沿触发
      • 三角表示沿边缘有效（转换时的竖线状态即为边缘）
        • 上升沿
          • 单独三角
        • 下降沿
          • 带圈三角

逻辑功能

  • $RS$触发器

    • 真值表

      R S $Q^{n+1}$
      0 0 不允许
      0 1 0
      1 0 1
      1 1 保持

    • 状态方程

      • $${Q^{n - 1}} = \overline S + RQ$$

    • 约束条件

      • $$R + S = 1$$
      • $$\overline R \overline S = 0$$
  • D触发器
    • 状态方程
      • $Q^{n+1}=D$
    • 约束条件
      • 上升沿有效

  • JK触发器

    • 真值表

      J K $Q^{n+1}$ 说明
      0 0 Q 保持
      0 1 0 置0
      1 0 1 置1
      1 1 $\bar Q$ 反转

    • 状态方程

      • $Q^{n+1}=J \bar Q + \bar KQ$

  • T触发器 ——由JK触发器衍生

    • 真值表

      T $Q^{n+1}$ 说明
      0 Q 保持
      1 $\bar Q$ 反转

    • 状态方程

      • $Q^{n+1}=T \bar Q + \bar TQ$
  • 写出状态方程
  • 根据$R_D$处理初始值
  • 有效边用虚线表示
  • 写出虚线处输入变量的值（以跳变前为准）
  • 写出输出的值，画出波形

时序逻辑电路分析

同步和异步的区别

知识点

  • 同步时序电路
    • 有统一的时钟
  • 异步时序电路
    • 有多个时钟

同步时序逻辑电路分析

知识点

  • $Moore$型

    • 输入输出没关系
    • 状态转移真值表与状态表相同

  • $Mealy$型

    • 输入输出有关系

    • 状态转移真值表与状态表不相同

    • $Q^{n+1}_1Q^{n+2}_2/Y$ $Q^{n+1}_1Q^{n+2}_2/Y$
      $Q_1Q_2/A$ 0 1
      00
      01
      10
      11
  • 状态转移图
    • 绘制
      • 写出状态转移表后假定一个为起点进行作画，依次转向下一个$n+1$的点，直到出现闭环
      • $Moore$格式为：
        • $Q_1Q_2/Z\stackrel{X}{\longrightarrow}$
        • $Z$为输出，$X$为输入
      • $Mealy$格式为：
        • $Q_1Q_2\stackrel{A/Y}{\longrightarrow}$
        • $Y$为输出，$A$为输入
      • 出现闭环后检查是否包含所有状态，若有未包含的状态，将其添加进入状态转移图
    • 注意
      • 对于计数器，环中包含了$n$个状态，则为$n$进制计数器
      • 只有一个环则具备自启动
      • 包含多个环则不具备自启动
  • 列写激励/驱动和输出方程
  • 导出状态方程
  • 列出状态转移真值表
  • 画状态转移表
  • 画状态转移图

集成计数器

常用集成计数器分析

  • 4位同步二进制计数器74161

    • 模值

      • 16

    • 计数过程

      • $0\to1\to2\to \cdots \to14\to15\to0$

    • 结构

      • $D_0D_1D_2D_3$,为输入位，其中$D_3$为高位
      • $Q_0Q_1Q_2Q_3$,为输出位，其中$Q_3$为高位
      • $RCO$,为进位标志，只有当计满16个数字时候才会更换电平
      • $PT$,为技术控制端
      • $$\overline {CLR} $$，为异步清零端
        • 一旦有效，则输出位全为0
      • $$\overline {LD} $$,为同步置数端
        • 一旦有效，输入等于输出

    • 功能表

    • 功能 实现条件
      加1计数 $$P = T = 1,\overline {CLR} = \overline {LD} = 1,CLK = \uparrow $$
      同步预置 $$P = T = 1,\overline {CLR} = 1,\overline {LD} = 0,CLK = \uparrow $$
      异步清零 $$\overline {CLR} = 0$$
      计数保持 $$P = 0,T = 1,\overline {CLR} = \overline {LD} = 1$$

  • 同步十进制计数器74160

    • 只有模与74161不同，为10

  • 4位同步二进制计数器74163

    • 只有清零与74161不同，为同步清零
      • $$\overline {CLR} = 0,CLK = \uparrow$$

常用集成计数器应用

知识点

  • 异步清零（存在过渡态）
    • $P = T = 1,\overline {LD} = 1,\overline {CLR}$接逻辑门输出时，计数初值一定为0，可以令$\overline{CLR}=0$的最小$Q$值-1，就是计数终值
    • 在设计中，1对应的输入值的非就是$\overline{CLR}$的逻辑表达式
    • 终值也等于模值
  • 同步置数
    • $P = T = 1,\overline {CLR} = 1,\overline {LD}$接逻辑门输出时，计数初值为$D_3D_2D_1D_0$，可以令$\overline{LD}=0$的最小$Q$值，就是计数终值
    • 在设计中，1对应的输入值的非就是$\overline{LD}$的逻辑表达式
    • 终值也等于$D_3D_2D_1D_0$+模值-1

移位寄存器应用

经过一个时钟脉冲左移一位，移出的数据删掉，缺位的数据补上串行输入的数据。

解题（移位寄存器）

集成逻辑门使用的实际问题

知识点

  • $TTL$
    • 低电平
      • 输入端接下拉电阻，阻值小于$$0.7k\Omega $$
      • 输入电压小于$0.4V$
    • 高电平
      • 输入端接下拉电阻，阻值大于$$0.7k\Omega $$
      • 输入电压大于$2.4V$
      • 悬空
  • $CMOS$
    • 低电平
      • 接近$0V$
    • 高电平
      • 接近电源电压
    • 不允许悬空

解题（集成逻辑门）

半导体存储器

知识点

  • $RAM$
    • 易失性
    • 相当于电脑的运行内存
    • 只用来暂存数据，无法长期存储
    • 一旦掉电数据损失
  • $ROM$
    • 非易失性
    • 相当于电脑的存储内存
    • 可以用来 长期存储
    • 掉电也不会造成数据损失
  • 容量
    • 存储容量=字数$\times$位数
      • 此处字数中的$K$并不是1000，而是$2^{10}$
    • $n$条地址线$\to2^n$个存储空间
    • $m$条数据线$\to m$的位数
  • 组合计算（小$RAM$组合大$RAM$）
    • 总需求=地址位需求$\times$数据位需求

解题（存储器）

555定时器

施密特触发器

知识点

  • 555定时器
    • 它可以方便地构成施密特触发器、单稳态触发器和多谐振荡器
  • 施密特触发器
    • 施密特触发器可以把不规则的信号整形成比较规则的数字脉冲信号。—波形整形
    • 端口
      • 2、6输入$u_i$
      • 4、8供电${V_{CC}}$
      • 1、5加入电容$(0.01\mu F)$后串联接地
      • 3输出$u_o$
    • $V_{T+}=\frac{2}{3}V_{CC}$
    • $V_{T-}=\frac{1}{3}V_{CC}$
    • 输出
      • 矩形波
      • 输入电压上升时与$V_T+$的交点处，由高变低
      • 输入电压下降时与$V_T-$的交点处，由低变高

解题（施密特触发器）

单稳态触发器

知识点

  • 高电平
    • 暂稳态
    • 输入电压小于$\frac{1}{3}V_{CC}$不触发
  • 低电平
    • 稳态
    • 输入电压大于$\frac{1}{3}V_{CC}$不触发
    • 宽度必须小于$T_W$
  • 暂稳态持续时间
    • $T_W=1.1RC$

解题（单稳态触发器）

多谐振荡器

知识点

  • 原理
    • 电容充放电
  • 电路内部自行产生输出电压
    • 波形发生器
  • 输出波形
    • $u_o$输出为矩形波
    • $C$输出为三角波
    • 三角波上升时，矩形波为高电平
    • 三角波下降时，矩形波为低电平
  • 持续时间
    • 充电路径
      • $V_{CC}\to C$
    • 高电平持续时间
      • $T_1=0.7(R_1+R_2)C$
      • 充电路径所经过的电阻和乘以电容的0.7倍
    • 放电路径
      • $C\to7$
    • 低电平持续时间
      • $T_2=0.7R_2C$
      • 放电路径所经过的电阻和乘以电容的0.7倍
    • 震荡周期
      • $T=T_1+T_2=0.7(R_1+2R_2)$

可编程逻辑器件

知识点

  • 可编程逻辑器件$PLD$是由与阵列和或阵列组成的，所以需要将逻辑表达式化为与或式
  • 通常使用最简与或式

解题（可编程逻辑器件）

A/D转换器

知识点

  • 采样$\to$保持$\to$量化$\to$编码
  • 分辨率
    • 转换器中能分辨输入模拟信号的最小量
    • 分辨率等于量程除以$2^n$,$n$为转换器位数
  • 输入与输出关系
    • $y=\frac{2^n-1}{V_{BEF}}x$
      • $V_{BEF}$为极限电压
      • $x$为输入电压
      • $n$为转换器位数
    • 最后将$y$化为2进制即可

解题（A/D转换器）

D/A转换器

知识点

  • 分辨率
    • 转换器中能输出输入模拟信号的最小量
    • 分辨率等于量程除以$2^n$,$n$为转换器位数
  • 输入与输出关系
    • $V_0=\frac {D-1}{2^n}V_{BEF}$
      • $V_{BEF}$为极限电压
      • $D$为二进制转为10进制的结果（输入的值）
      • $n$为转换器位数
