半导体器件基础

半导体基础

知识点

半导体

  • 本征半导体
  • 杂志半导体
    • P型半导体
      • 形成
        • 加入三价元素$B,Al$
      • 特性
        • 空穴多，自由电子少
    • N型半导体
      • 形成
        • 加入五价元素$P，As$
      • 特性
        • 自由电子多，空穴少

PN结

  • 正偏
    • $P+,N-\to$$P$点电势高，$N$点电势低
    • 扩散电流大于漂移电流
    • $PN$结变窄，导通
  • 反偏
    • $P-,N+\to$$P$点电势低，$N$点电势高
    • 扩散电流小于漂移电流
    • $PN$结变宽，截止

二极管

知识点

  • 晶体二极管

    • 单向导电性

    • 伏安特性

      • | | | | |
        | —— | :———: | :————: | :—————: |
        | | 正向特性 | 正向特性 | 反向特性 |
        | 材料 | 死区电压 | 导通电压 | 反向饱和电流 |
        | $Si$ | $0-0.5V$ | $0.6-0.8V$ | $nA$级 |
        | $Ge$ | $0-0.1V$ | $0.2-0.3V$ | $\mu A$级 |

  • 稳压二极管

    • 伏安特性
      • 在正向电压加载的时候和普通二级管相同
      • 反向电压小加载时饱和电流几乎为0
      • 反向电压大加载时称为反向击穿区——工作区
        • 输出的值即为稳压状态的值

  • 判断二极管正负极

  • 确定二极管两端电压的大小
  • 确定二极管状态（导通/截止）
  • 画出波形

解题（二极管）

三极管

知识点

  • 简介
    • 符号介绍
      • 集电区/集电极$c\to Collector$
      • 基区/基极$b\to Base$
      • 发射区/发射极$e\to Emission$
    • 集电结
      • 集电极与基极接触的地方
    • 发射结
      • 发射极与基极接触的地方
    • 状态
      • $P>N$为正偏，$P<N$为反骗
      • 放大状态
        • 三极管正常工作的区间
        • 发射结正偏、 集电结反偏
        • 发射区掺杂浓度高，基区薄且掺杂浓度低，集电区面积大。
      • 饱和状态
        • 发射结、集电结正偏
      • 截止状态
        • 发射结、集电结反偏
    • 双极性晶体管
      • 电流放大作用
      • $i_c=\beta i_b$，$i_b$控制$i_c$，电流控制型

  • 分类

    • $PNP$型
      • 箭头向内
      • $e>b>c$
    • $NPN$型
      • 箭头向外
      • $c>b>e$

  • 给出三个级的电压时，中间电压为$b$，离$b$近的电压为$e$，离$b$远的电压为$c$

  • 温度升高使正向特性曲线左移，正向压降减小；反向特性曲线下移，反向电流增大。

  • 复合管（达林顿管）

    • 原则
      • 保证前后级电流方向一致
    • 类型
      • 一入两出为$PNP$
      • 两入一出为$NPN$
    • 放大系数
      • $\beta_=\beta_1\beta_2$

解题（三极管）

场效应管（$FET$)

知识点

  • 简介
    • 符号介绍
      • 源极$S\to Source$
      • 栅极$G\to Grid$
      • 漏极$D\to Drain$
    • 单极性器件
      • 电流放大作用
      • $i_D=f(u_{GS})$，$u_{GS}$控制$i_D$，电压控制型
  • 原理
    • 控制三个极之间的电压形成沟道
  • 分类
    • $P$沟道为电流从场效应管中流出
    • $N$沟道为电流从场效应管外流入
    • 结型（$JFET$）
    • 绝缘栅场（$MOS$）
      • 增强型
        • $U_{GS}=0$时，沟道不存在
      • 耗尽型
        • $U_{GS}=0$时，沟道已存在
  • 转移特性曲线

解题（场效应管）

基本放大电路

放大电路基础

知识点

  • 标注

    • 直流信号
      • 全部大写
      • $I_B,U_{CE}$
    • 交流信号
      • 全部小写
      • $i_b,u_{ce}$
    • 交直流信号
      • 上标小写，下标大写
      • $i_B,u_{CE}$

  • 组成

    • 三极管
      • $T$起放大作用
    • 偏置电路
      • $R_b,V_{CC}$，使二极管工作于放大区，给输出信号提供能量
    • 集电极负载电阻
      • $R_c$，将集电极电流转换为电压变化
      • $U_{CE}=V_{CC}-I_CR_C$
    • 耦合电容
      • $C_1,C_2$隔直流通交流
  • 符号
    • $i_B$基极电流
    • $i_C$集电极电流
    • $u_{CE}$集电极和发射极间的电压
  • 输出特性曲线

  • 直流负载线

    • 对应的每一个交点为静态工作点
      • $I_{BQ}=\frac{V_{CC}-U_{BEQ}}{R_b}$
      • $I_{CQ}=\beta I_{BQ}$静态工作点
      • 用于分析静态工作偏高或偏低造成的失真现象
        • 工作点过高：波谷失真为底部失真/饱和失真
        • 工作点过低：波峰失真为顶部失真/截止失真
        • 输入信号幅度过大导致饱和与截止为双向失真

  • 组态

    • 共射
      • 输入输出反相
      • 电压、电流放大$\to$中间级
      • 输入电阻与输出电阻中等
    • 共基
      • 输入输出同相
      • 电压放大
      • 电流不放大$\to$恒流源电路
      • 输入电阻最小、输出电阻最大
    • 共集
      • 输入输出反相
      • 电流放大
      • 电压不放大$ \to$缓冲级
      • 输入电阻最大$\to$获取信号能力强$\to$输入级
      • 输出电阻最小$\to$带负载能力强$\to$输出级

解题

放大电路的分析方法

知识点

  • 静态分析
    • 画直流通路(电容视为断路，电感视为短路)
  • 求静态工作点$Q$
    • $I_{BQ},I_{CQ},U_{CEQ}$
  • 动态分析
    • 画微变等效电路(把电容、直流电源视为短路)
    • 三极管微变等效模型
      • $be$间等效为$r_{be}$的电阻，电流为$i_b$
        • $r_{be}=r_{bb}’+(1+\beta)\frac {26mV}{I_{EQ} }$
        • 题目未明确说明时$r_{bb}’=200\Omega$
      • $ce$间等效为$\beta i_b$的受控源，电流为$i_c$
  • 求动态参数
    • $A_u,R_o,R_i$
    • 普通放大电路
      • $A_u=\frac{u_o}{u_i}=\frac{-i_c(R_c//R_L)}{i_br_{be} }=-\beta \frac{R_c//R_L}{r_{be} }$
      • 从输入往输出方向看
        • $R_i=R_b//r_{be}$
      • 将外接负载断开，信号源短路，从输出往输入方向看
        • $R_o=R_c$
    • 分压式偏置放大电路
      • 静态工作点
        • $U_{BQ}=\frac{R_{b2}}{R_{b1}+R_{b2}}V_{CC}$
        • $I_{EQ}=\frac{U_{BQ}-U_{BEQ}}{R_e}$
        • $I_{CQ} \approx I_{EQ} $
        • $I_{BQ}=\frac{I_{CO}}{\beta}$
        • $U_{CEQ}=V_{CC}-I_{CQ}(R_c+R_e)$
      • 动态等效电路
        • $A_u=\frac{u_o}{u_i}=\frac{-i_c(R_c//R_L)}{i_br_{be} }=-\beta \frac{R_c//R_L}{r_{be} }$
        • $R_i=R_{b_1}//R_{b_2}//R_{b3}$
        • $R_o=R_c$

解题

多级放大电路

多级放大电路

  • 第一级集电极与第二级基极相连接

  • 直接耦合

    • 直接连接
  • 阻容耦合
    • 连接中加上电容

  • 变压器耦合

    • 连接中加上变压器

  • 总增益

    • 对数增益
      • $A=20\lg A_n=20\lg A_{n1}+\cdots$
      • 单位为$dB$
    • $A_u=A_{u_1}A_{u2}\cdots$
  • 输入电阻
    • $R_i=R_{i1}$
  • 输出电阻
    • $R_o=R_{on}$
  • 频率
    • $f_L>$任意单级放大电路的下限截止频率
    • $f_H<$任意单级放大电路的上限截止频率

差动放大电路

知识点

  • 参数理想对称
    • $R_{b1}=R_{b2},R_{c1}=R_{c2},R_{e1}=R_{e2}$
    • $T_1,T_2$在任何温度下特性均相同
  • 作用
    • 解决因为温度影响，输入为0时，集电极有电流输出
    • 由于输出为二者的结果之差（共模信号），就可以抑制零漂（温漂）
  • 共模信号
    • 大小相同，方向相同
    • $u_{ic}=u_{i1}+u_{i2}$
    • $A_{uc}=\frac{u_{oc} }{u_{ic} }=\frac{u_{oc} }{u_{i1}+u_{i2} }$
  • 差模信号
    • 大小相同，方向相反（增益为负数）
    • $u_{id}=u_{i1}-u_{i2}$
    • $A_{ud}=\frac{u_{od} }{u_{id} }=\frac{u_{od} }{u_{i1}-u_{i2} }$

  • 共模抑制比

    • $K_{CMR}=|\frac{A_{ud} }{A_{uc} }|$
    • $K_{CMR}$越大，电路放大性能越好

  • 射极电阻$R_e$越大，抑制零漂效果越好，但需要负电源$u_{EE}$越高

解题（差动放大电路）

集成运算放大器的线性应用

运算电路(同相、反相、差分、跟随)

知识点

  • 符号
    • 同相输入端
      • $u_+$
    • 反相输入端
      • $u_-$
    • 输出端
      • $u_o$
    • 开环差模放大倍数
      • $A_{od}$
      • 默认为无穷

  • 线性区
    • $u_o=A_{od}(u_+-u_-)$
    • 输出电压随输入电压线性变化
    • 集成运算电路、滤波电路

  • 非线性区

    • $u_o=+u_{OM}或-u_{OM}$
    • 电压比较器

  • 内部组成

    • 输入级
      • 差动放大电路$\to$抑制零漂
    • 中间级
      • 共射放大电路$\to$提高电压放大倍数
    • 输出级
      • 互补对称输出级电路$\to$提高带负载能力
    • 偏执电路
      • 电流源电路$\to$提供静态偏置电流

  • 集成运放的两个工作区域

    • 线性工作区
      • 负反馈
      • 虚短
        • $u_+=u_-$
      • 虚断
        • $i_+=i- \approx 0$
    • 非线性工作区
      • 正反馈或开环

基本运算电路

  • 反相比例运算电路
    • $u_o=-\frac{R_f}{R_1}u_i$
  • 同相比例运算电路
    • $u_o=(1+\frac{R_f}{R_1})u_i$
  • 电压跟随器
    • 具有输入阻抗高、输出阻抗低的特点，可以起到阻抗匹配的作用，能够使得后一级的放大电路更好的工作。
    • 隔离，把电路置于前级和功放之间，可以切断后一级的反电动势对前级的干扰作用。
    • 由同相比例运算电路中的$R_f,R_1$进行极限取值得到
  • 差分比例电路（减法运算）
    • 运用叠加原理得到
      • $u_o=u_o’+u_o’’=-\frac{R_f}{R_1}u_{i1}+(1+ \frac{R_f}{R_1})\frac{R_3}{R_2+R_3}u_{i2}$

积分、微分运算电路

知识点

  • 电容

  • 积分电路

    • $u_o=-\frac{1}{RC}\int{u_i dt}$
    • 可将正弦改为余弦，方波改为三角波

  • 微分电路

    • $u_o=-RC\frac{du_i}{dt}$

  • 模拟乘法器

    • 输出为输入信号相乘
      • $u_o=Ku_xu_y$
    • 接入反馈电路后，当$u_o$作为其中一个输入时，可以实现除法电路

电压比较器

知识点

  • $u_+>u_-\to u_o=+u_{om}$

  • $u_+<u_-\to u_o=-u_{om}$

  • 门限电压

    • $u_{TH},u_{TL}$
  • 回差电压
    • $\Delta u_{TH}=u_{TH}-u_{TL}$
  • 传输特性曲线图
    • 稳压二极管输出为$y$轴
    • 两个门限电压为$x$轴
    • 输入从负无穷增加时一个带有箭头的折线
    • 输入从正无穷减小时一个带有箭头的折线

解题（电压比较器）

频率特性

放大电路的频率响应

知识点

  • 电压放大倍数与信号频率的关系

    • $A_u=\frac{A_u(f)}{\varphi(f) }$

    • 幅频响应，电压放大倍数的模与信号频率的关系

      • $A_u(f)$

    • 相频响应，放大电路输出电压与输入电压的相位差与信号频率的关系

      • ${\varphi(f) }$

  • 低频区

    • 下限截止频率$f_L$
      • 信号上升到$0.707A_{um}$
    • 下限截止频率之下就是低频区
    • 放大倍数下降到中频的0.7倍
    • 增益下降$3dB$
    • 隔直耦合电容、旁路电容，阻抗增大，使信号衰减，增益下降。

  • 高频区

    • 上限截止频率$f_H$
      • 信号下降到$0.707A_{um}$
    • 上限截止频率之上就是高频区
    • 放大倍数下降到中频的0.7倍
    • 增益下降$3dB$
    • 并接的极间电容和杂散电容容抗减小，对信号分流，增益下降。

  • 中频区

    • $BW=f_H-f_L$
    • 上下限截止频率中间的频率带
    • 忽略所有电容的影响，视隔直电容、旁路电容为交流短路，视极间电容和杂散电容为交流开路。

有源滤波电路

知识点

  • 工作在线性区
  • 组成
    • 放大电路
    • $RC$网络
  • 分类
    • $A$带阻滤波器
      • 允许高频和低频
    • $B$低通滤波器
      • 允许低频
      • 电容在输入支路进行高频阻断
    • $C$带通滤波器
      • 允许中频
    • $D$高通滤波器
      • 允许高频
      • 电容在输入主路进行低频截断
  • 特征频率
    • $f_0=\frac{1}{2\pi RC }$

解题（有源滤波）

反馈放大电路

反馈的判断

知识点

  • 将输出量（输出电压或输出电流）的一部分或全部通过一定的电路形式作用到输入回路，用来影响其输入量（放大电路的输入电压或输入电流）的措施。放大电路无反馈时又称为开环，有反馈时又称为闭环。

  • 一般表达式

    • 开环放大倍数
      • $A=\frac{X_o}{X_{id}}$
    • 反馈系数
      • $F=\frac{X_f}{X_o}$
    • 闭环放大倍数（反馈深度）
      • $A_f=\frac{A}{1+AF}$

  • 反馈判断

    • 由输出支路决定
      • 电压反馈
        • 反馈支路和输出集中于一个点
      • 电流反馈
        • 反馈支路和输出不在一个点
    • 由输入支路决定
      • 串联反馈
        • 反馈支路与输入不在一个点
      • 并联反馈
        • 反馈支路和输入在一个点

  • 瞬时极性法

    • $u_id=u_+-u_-$

    • 假设输入的极性，判断反馈的极性

    • 如果会造成原有量减少则为负反馈

      • 极性相反在同一点
      • 极性相同在不同点

    • 如果会造成原有量增加则为正反馈

      • 极性相反在不同点
      • 极性相同在同一点

    • 三极管

      • $i_e=i_b+i_c$

    • | 组态 | 结果 |
      | :—————: | :—————————————————————————————: |
      | 共射极电路 | $bc$的瞬时极性相反，
      当有射极电阻并且没有旁路电容时，
      $be$瞬时极性相同。 |
      | 共基极电路 | $ce$瞬时极性相同，
      当有基极电阻无旁路电容时，
      $be$瞬时极性相反。 |
      | 共集电极电路 | $be$瞬时极性相同
      $bc$瞬时极性相反。 |

反馈的作用

  • 直流负反馈
    • 稳定静态工作点
  • 电压负反馈
    • 减小$R_o$
    • 提高带负载能力，稳定输出电压
  • 电流负反馈
    • 增大$R_o$
    • 稳定输出电流，
  • 串联负反馈
    • 增大$R_i$
    • 减小电路向信号源索取的电流
  • 并联负反馈
    • 减小$R_i$

解题（反馈）

深度负反馈

知识点

  • 在深度负反馈条件下，闭环放大倍数近似等于反馈系数的倒数，与有源器件的参数基本无关。一般反馈网络是无源元件构成的，其隐定性优于有源器件，因此深度负反馈时的放大倍数比较稳定。
  • $A_f=\frac{A}{1+AF}$
    • 当$1+AF\gg 1$时，$A_f\approx \frac{1}{F}$
  • 串联负反馈
    • $u_i \approx u_f$
  • 并联负反馈
    • $i_i \approx i_f$

功率放大电路

功率放大电路基础问题

知识点

  • 分类

    • 甲类
      • 三极管构成的一个基本的放大电路
        • $Q$点在$i_c-u_{CE} $中部
        • 导通角$\theta=360^\circ \to$形成完整的正弦波
        • 管耗大，效率低
    • 乙类
      • 两个三极管$NPN$和$PNP$
        • $Q$点在$i_c-u_{CE} $的$u_{CE}$位置
        • 导通角$\theta=180^\circ \to$半个正弦波$\to$出现死区电压$\to$交越失真
        • 管耗小，效率高
      • $OTL$
        • 输出端包含电容
        • 无输入变压器的功率放大电路
      • $OCL$
        • 输出端无电容
        • 输出无电容的功率放大电路

    • 甲乙类
      • 在乙类的两个三极管之间加入两个二极管
      • $Q$点在$i_c-u_{CE} $中部偏下的位置
      • 导通角$180^\circ<\theta<360^\circ \to$多半个正弦波$\to$消除交越失真
      • 效率较高
